# DT Money

Aplicação desenvolvida durante o Chapter II do Ignite na trilha de ReactJS, com o objetivo de aplicar o conteúdo estudado sobre:

- Styled Components
- MirageJS
- Context API
- Hooks
- Axios
- Entre outros.

## Execução

1. Clone o repositório

```sh
git clone https://github.com/abigailarruda/dt-money.git
```

2. Instale os pacotes

```sh
yarn
```

3. Execute a aplicação

```sh
yarn start
```
